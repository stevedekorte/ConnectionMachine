ConnectionMachine.fun
/ about
A CM2 inspired 32x32 LED panel wall art project.
Design files and source code available on github.
Board design by Jonathan Foote.
design
Gerbers PCB files
rendering (one of two 16x32 LED panels)
board
1 × BCM2835 - Rhaspberry PI 0 CPU chip
??? × HT1632C - LED driver chip
1024 × ??? LEDs
1 × ??? power supply
High-side buffer research, assume ULN2003 500mA low side:
HEF40244B -- -25-35 mA
74ACT11240 -- -25mA, max -50mA
74VHC244 -- -25 mA
SN74AHC244 -- 2003 datasheet say-25mA, website says -50mA
SN74LV244A-EP -35mA Pinout -- yuck!
SN74ABT541B
CY74FCT16244T 16 channel, max -60mA, expensive
SN74ABT541B 8 channel, -32mA, good pinout
Discrete P channel FETs:
AOSS21115C
AO3400
FDG3616 -2x P-channel 700mA
case
sides: 26-1/2" x 26-1/2" (outside) modern deep black wood from pictureframes.com
front: Chemcast black LED acrylic, 1/8" thick with machined edges from tapplastics.com
rear: PVC dark gray, 1/8" thick with machined edges from tapplastics.com
misc: PCB standoffs, mounting bracket, wood screws, felt pads, and power cord from amazon.com
server
Python3 secure web sockets server
receives json with hex framebuffer and brightness
sends to HT1632C Python3 binding
